# 💱 Convertly - Currency Converter

Convertly is a modern and responsive Currency Converter built with HTML, CSS, and JavaScript. It allows users to convert currencies using real-time exchange rates while providing a clean user interface with Dark and Light theme support.

## 🚀 Features

* Real-time currency conversion
* Support for 20+ international currencies
* Dark & Light theme toggle
* Dynamic currency symbol display
* Responsive design for desktop and mobile devices
* Smooth UI animations
* Input validation
* Error handling for API requests

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
